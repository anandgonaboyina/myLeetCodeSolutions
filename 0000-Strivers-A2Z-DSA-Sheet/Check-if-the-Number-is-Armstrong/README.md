<h2><a href="https://takeuforward.org/plus/">Check if the Number is Armstrong</a></h2><h3>Medium</h3><hr><p>You are given an integer <strong>n</strong>. You need to check whether it is an <strong>armstrong </strong>number or not. Return <strong>true </strong>if it is an <strong>armstrong </strong>number, otherwise return <strong>false</strong>.</p><p><br></p><p>An <strong>armstrong </strong>number is a number which is <strong>equal </strong>to the <strong>sum </strong>of the <strong>digits </strong>of the number, raised to the <strong>power</strong> of the <strong>number </strong>of digits.</p><br><br><p class="tuf-text-14">Example 1</p><div class="tuf-example text-new_secondary"><p><strong>Input</strong>: n = 153</p><p><strong>Output</strong>: true</p><p><strong>Explanation</strong>: Number of digits : 3.</p><p>1<sup>3</sup> + 5<sup>3</sup> + 3<sup>3</sup> = 1 + 125 + 27 = 153.</p><p>Therefore, it is an Armstrong number.</p></div><br><br><p class="tuf-text-14 ">Example 2</p><div class="tuf-example text-new_secondary"><p><strong>Input</strong>: n = 12</p><p><strong>Output</strong>: false</p><p><strong>Explanation</strong>: Number of digits : 2.</p><p>1<sup>2</sup> + 2<sup>2</sup> = 1 + 4 = 5.</p><p>Therefore, it is not an Armstrong number.</p></div><br><br><p class="tuf-text-14 ">Constraints</p><div class="tuf-dark-content-box"><ul><li>0 &lt;= n &lt;= 10<sup>9</sup></li></ul></div><br><br>
