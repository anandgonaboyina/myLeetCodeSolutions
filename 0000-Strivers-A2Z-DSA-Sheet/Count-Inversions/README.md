<h2><a href="https://takeuforward.org/plus/">Count Inversions</a></h2><h3>Medium</h3><hr><p>Given an integer array <strong>nums</strong>. Return the <strong>number </strong>of <strong>inversions </strong>in the array.</p><p><br></p><p>Two elements <strong>a[i]</strong> and <strong>a[j] </strong>form an <strong>inversion </strong>if <strong>a[i] &gt; a[j]</strong> and <strong>i &lt; j</strong>.</p><p><br></p><ul><li>It indicates how <strong>close </strong>an array is to being <strong>sorted</strong>.</li></ul><p><br></p><ul><li>A <strong>sorted </strong>array has an inversion count of <strong>0</strong>.</li></ul><p><br></p><ul><li>An array <strong>sorted </strong>in <strong>descending </strong>order has <strong>maximum </strong>inversion.</li></ul><br><br><p class="tuf-text-14">Example 1</p><div class="tuf-example text-new_secondary"><p><strong>Input</strong>: nums = [2, 3, 7, 1, 3, 5]</p><p><strong>Output</strong>: 5</p><p><strong>Explanation</strong>:</p><p>The responsible indexes are:</p><p>nums[0], nums[3], values: 2 &gt; 1 &amp; indexes: 0 &lt; 3</p><p>nums[1], nums[3], values: 3 &gt; 1 &amp; indexes: 1 &lt; 3</p><p>nums[2], nums[3], values: 7 &gt; 1 &amp; indexes: 2 &lt; 3</p><p>nums[2], nums[4], values: 7 &gt; 3 &amp; indexes: 2 &lt; 4</p><p>nums[2], nums[5], values: 7 &gt; 5 &amp; indexes: 2 &lt; 5</p></div><br><br><p class="tuf-text-14 ">Example 2</p><div class="tuf-example text-new_secondary"><p><strong>Input</strong>: nums = [-10, -5, 6, 11, 15, 17]</p><p><strong>Output</strong>: 0</p><p><strong>Explanation</strong>:</p><p>nums is sorted, hence no inversions present.</p></div><br><br><p class="tuf-text-14 ">Constraints</p><div class="tuf-dark-content-box"><ul><li>1 &lt;= nums.length &lt;= 10<sup>5</sup></li><li>-10<sup>5</sup> &lt;= nums[i] &lt;= 10<sup>5</sup></li></ul></div><br><br>
