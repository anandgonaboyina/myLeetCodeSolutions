<h2><a href="https://takeuforward.org/plus/">Find Nth root of a number</a></h2><h3>Medium</h3><hr><p>Given two numbers N and M, find the <strong>Nth</strong> root of M. The Nth root of a number M is defined as a number X such that when X is raised to the power of N, it equals M. If the Nth root is not an integer, return -1.</p><br><br><p class="tuf-text-14">Example 1</p><div class="tuf-example text-new_secondary"><p>Input: N = 3, M = 27</p><p>Output: 3</p><p>Explanation: The cube root of 27 is equal to 3.</p></div><br><br><p class="tuf-text-14 ">Example 2</p><div class="tuf-example text-new_secondary"><p>Input: N = 4, M = 69</p><p>Output:-1</p><p>Explanation: The 4th root of 69 does not exist. So, the answer is -1.</p></div><br><br><p class="tuf-text-14 ">Constraints</p><div class="tuf-dark-content-box"><ul><li>&nbsp;&nbsp;1 &lt;= N &lt;= 30</li><li>&nbsp;&nbsp;1 &lt;= M &lt;= 10<sup>9</sup></li></ul></div><br><br>
