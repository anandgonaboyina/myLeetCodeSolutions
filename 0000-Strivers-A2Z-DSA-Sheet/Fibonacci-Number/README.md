<h2><a href="https://takeuforward.org/plus/">Fibonacci Number</a></h2><h3>Medium</h3><hr><p>The <strong>Fibonacci</strong> numbers, commonly denoted F(n) form a sequence, called the Fibonacci sequence, such that each number is the sum of the two preceding ones, starting from 0 and 1. That is,</p><p><br></p><ul><li>F(0) = 0, F(1) = 1</li><li>F(n) = F(n - 1) + F(n - 2), for n &gt; 1.</li></ul><p><br></p><p>Given <strong>n</strong>, calculate <strong>F(n)</strong>.</p><br><br><p class="tuf-text-14">Example 1</p><div class="tuf-example text-new_secondary"><p><strong>Input :</strong> n = 2</p><p><strong>Output :</strong> 1</p><p><strong>Explanation :</strong> F(2) = F(1) + F(0) =&gt; 1 + 0 =&gt; 1.</p></div><br><br><p class="tuf-text-14 ">Example 2</p><div class="tuf-example text-new_secondary"><p><strong>Input :</strong> n = 3</p><p><strong>Output :</strong> 2</p><p><strong>Explanation :</strong> F(3) = F(2) + F(1) =&gt; 1 + 1 =&gt; 2.</p></div><br><br><p class="tuf-text-14 ">Constraints</p><div class="tuf-dark-content-box"><ul><li>0 &lt;= n &lt;= 20</li></ul></div><br><br>
