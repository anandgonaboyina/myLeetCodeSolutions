# Sort an array of 0's 1's and 2's

## Problem Statement
<p>Given an array <strong>nums</strong> consisting of only 0, 1, or 2. Sort the array in non-decreasing order. </p><p><br></p><p>The sorting must be done in-place, without making a copy of the original array.</p>

## Examples
<p class="tuf-text-14">Example 1</p><div class="tuf-example text-new_secondary"><p><strong>Input</strong>: nums = [1, 0, 2, 1, 0]</p><p><strong>Output</strong>: [0, 0, 1, 1, 2]</p><p><strong>Explanation</strong>: </p><p>The nums array in sorted order has 2 zeroes, 2 ones and 1 two</p></div><br><br><p class="tuf-text-14 ">Example 2</p><div class="tuf-example text-new_secondary"><p><strong>Input</strong>: nums = [0, 0, 1, 1, 1]</p><p><strong>Output</strong>: [0, 0, 1, 1, 1]</p><p><strong>Explanation</strong>: </p><p>The nums array in sorted order has 2 zeroes, 3 ones and zero twos</p></div><br><br><p class="tuf-text-14 ">Constraints</p><div class="tuf-dark-content-box"><ul><li>1 &lt;= nums.length &lt;= 10<sup>5</sup></li><li>nums consists of 0, 1 and 2 only.</li></ul></div><br><br>
