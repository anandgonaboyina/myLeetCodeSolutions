<h2><a href="https://takeuforward.org/plus/">Longest subarray with sum K</a></h2><h3>Medium</h3><hr><p>Given an array nums of size n and an integer k, find the length of the <strong>longest sub-array</strong> that sums to k. If no such sub-array exists, return 0.</p><br><br><p class="tuf-text-14">Example 1</p><div class="tuf-example text-new_secondary"><p><strong>Input</strong>: nums = [10, 5, 2, 7, 1, 9],&nbsp;&nbsp;k=15</p><p><strong>Output</strong>: 4</p><p><strong>Explanation</strong>: </p><p>The longest sub-array with a sum equal to 15 is [5, 2, 7, 1], which has a length of 4. This sub-array starts at index 1 and ends at index 4, and the sum of its elements (5 + 2 + 7 + 1) equals 15. Therefore, the length of this sub-array is 4.</p></div><br><br><p class="tuf-text-14 ">Example 2</p><div class="tuf-example text-new_secondary"><p><strong>Input</strong>: nums = [-3, 2, 1],&nbsp;k=6</p><p><strong>Output</strong>: 0</p><p><strong>Explanation</strong>: </p><p>There is no sub-array in the array that sums to 6. Therefore, the output is 0.</p></div><br><br><p class="tuf-text-14 ">Constraints</p><div class="tuf-dark-content-box"><ul><li>&nbsp;1&lt;=n&lt;=10<sup>5</sup></li><li>&nbsp;-10<sup>5</sup>&lt;=nums[i]&lt;=10<sup>5</sup></li><li>&nbsp;-10<sup>9</sup>&lt;= k&lt;=10<sup>9</sup></li></ul></div><br><br>
