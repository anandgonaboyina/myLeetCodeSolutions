<h2><a href="https://takeuforward.org/plus/">Check if the Array is Sorted II</a></h2><h3>Medium</h3><hr><p>Given an array <strong>nums</strong> of n integers, return true if the array nums is sorted in non-decreasing order or else false.</p><br><br><p class="tuf-text-14">Example 1</p><div class="tuf-example text-new_secondary"><p><strong>Input :</strong> nums = [1, 2, 3, 4, 5]</p><p><strong>Output :</strong> true</p><p><strong>Explanation :</strong> For all i (1 &lt;= i &lt;= 4) it holds nums[i] &lt;= nums[i+1], hence it is sorted and we return true.</p></div><br><br><p class="tuf-text-14 ">Example 2</p><div class="tuf-example text-new_secondary"><p><strong>Input :</strong> nums = [1, 2, 1, 4, 5]</p><p><strong>Output :</strong> false</p><p><strong>Explanation :</strong> For i == 2 it does not hold nums[i] &lt;= nums[i+1], hence it is not sorted and we return false.</p></div><br><br><p class="tuf-text-14 ">Constraints</p><div class="tuf-dark-content-box"><ul><li>1 &lt;= n &lt;= 100</li><li>1 &lt;= nums[i] &lt;= 100</li></ul></div><br><br>
