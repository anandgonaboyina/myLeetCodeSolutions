<h2><a href="https://takeuforward.org/plus/">Floor and Ceil in Sorted Array</a></h2><h3>Medium</h3><hr><p>Given a sorted array nums and an integer x. Find the <strong>floor</strong> and <strong>ceil</strong> of x in nums. The floor of x is the largest element in the array which is smaller than or equal to x. The ceiling of x is the smallest element in the array greater than or equal to x. If no floor or ceil exists, output -1.</p><br><br><p class="tuf-text-14">Example 1</p><div class="tuf-example text-new_secondary"><p>Input : nums =[3, 4, 4, 7, 8, 10], x= 5</p><p>Output: 4 7</p><p>Explanation: The floor of 5 in the array is 4, and the ceiling of 5 in the array is 7.</p></div><br><br><p class="tuf-text-14 ">Example 2</p><div class="tuf-example text-new_secondary"><p>Input : nums =[3, 4, 4, 7, 8, 10], x= 8</p><p>Output: 8 8</p><p>Explanation: The floor of 8 in the array is 8, and the ceiling of 8 in the array is also 8.</p></div><br><br><p class="tuf-text-14 ">Constraints</p><div class="tuf-dark-content-box"><ul><li>&nbsp;&nbsp;1 &lt;= nums.length &lt;= 10<sup>5</sup>&nbsp;</li><li>&nbsp;&nbsp;0 &lt; nums[i], x &lt; 10<sup>5&nbsp;</sup></li><li>&nbsp;&nbsp;nums is sorted in ascending order.</li></ul></div><br><br>
