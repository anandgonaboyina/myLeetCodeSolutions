<h2><a href="https://takeuforward.org/plus/">Leaders in an Array</a></h2><h3>Medium</h3><hr><p>Given an integer array <strong>nums</strong>, return a list of all the leaders in the array.</p><p><br></p><p>A leader in an array is an element whose value is <strong>strictly</strong> greater than all elements to its right in the given array. The rightmost element is always a leader. The elements in the leader array must appear in the <strong>order they appear</strong> in the nums array.</p><br><br><p class="tuf-text-14">Example 1</p><div class="tuf-example text-new_secondary"><p><strong>Input</strong>: nums = [1, 2, 5, 3, 1, 2]</p><p><strong>Output</strong>: [5, 3, 2]</p><p><strong>Explanation</strong>: </p><p>2 is the rightmost element, 3 is the largest element in the index range [3, 5], 5 is the largest element in the index range [2, 5]</p></div><br><br><p class="tuf-text-14 ">Example 2</p><div class="tuf-example text-new_secondary"><p><strong>Input</strong>: nums = [-3, 4, 5, 1, -4, -5]</p><p><strong>Output</strong>: [5, 1, -4, -5]</p><p><strong>Explanation</strong>: </p><p>-5 is the rightmost element, -4 is the largest element in the index range [4, 5], 1 is the largest element in the index range [3, 5] and 5 is the largest element in the range [2, 5]</p></div><br><br><p class="tuf-text-14 ">Constraints</p><div class="tuf-dark-content-box"><ul><li>1 &lt;= nums.length &lt;= 10<sup>5</sup></li><li>-10<sup>4</sup> &lt;= nums[i] &lt;= 10<sup>4</sup></li></ul></div><br><br>
