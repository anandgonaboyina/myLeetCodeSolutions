<h2><a href="https://takeuforward.org/plus/">Pascal's Triangle I</a></h2><h3>Medium</h3><hr><p>Given two integers<strong> r</strong> and <strong>c</strong>, return the value at the <strong>r</strong><sup><strong>th </strong></sup>row and <strong>c</strong><sup><strong>th</strong></sup><strong> </strong>column (1-indexed) in a Pascal's Triangle.</p><p><br></p><p>In <strong>Pascal's triangle</strong>:</p><p><br></p><ul><li>The first row contains a single element 1.</li></ul><p><br></p><ul><li>Each row has one more element than the previous row.</li></ul><p><br></p><ul><li>Every row starts and ends with 1.</li></ul><p><br></p><p><strong>For all interior elements (i.e., not at the ends), the value at position (r, c) is computed as the sum of the two elements directly above it from the previous row:</strong></p><pre class="ql-syntax" spellcheck="false">Pascal[r][c]=Pascal[r−1][c−1]+Pascal[r−1][c]
</pre><ul><li>where indexing is 1-based</li></ul><br><br><p class="tuf-text-14">Example 1</p><div class="tuf-example text-new_secondary"><p><strong>Input</strong>: r = 4, c = 2</p><p><strong>Output</strong>: 3</p><p><strong>Explanation</strong>: </p><p>The Pascal's Triangle is as follows:</p><p>1</p><p>1 1</p><p>1 2 1</p><p>1 3 3 1</p><p>....</p><p>Thus, value at row 4 and column 2 = 3</p></div><br><br><p class="tuf-text-14 ">Example 2</p><div class="tuf-example text-new_secondary"><p><strong>Input</strong>: r = 5, c = 3</p><p><strong>Output</strong>: 6</p><p><strong>Explanation</strong>: </p><p>The Pascal's Triangle is as follows:</p><p>1</p><p>1 1</p><p>1 2 1</p><p>1 3 3 1</p><p>1 4 6 4 1</p><p>....</p><p>Thus, value at row 5 and column 3 = 6</p></div><br><br><p class="tuf-text-14 ">Constraints</p><div class="tuf-dark-content-box"><ul><li>1 &lt;= r, c &lt;= 30</li><li>c &lt;= r</li><li>All values will fit inside a 32-bit integer.</li></ul></div><br><br>
