<h2><a href="https://takeuforward.org/plus/">Find out how many times the array is rotated</a></h2><h3>Medium</h3><hr><p>Given an integer array nums of size n, sorted in ascending order with distinct values. The array has been <strong>right</strong> <strong>rotated</strong> an unknown number of times, between 0 and n-1 (including). Determine the <strong>number</strong> of rotations performed on the array.</p><br><br><p class="tuf-text-14">Example 1</p><div class="tuf-example text-new_secondary"><p>Input : nums = [4, 5, 6, 7, 0, 1, 2, 3]</p><p>Output: 4</p><p>Explanation: The original array should be [0, 1, 2, 3, 4, 5, 6, 7]. So, we can notice that the array has been rotated 4 times.</p></div><br><br><p class="tuf-text-14 ">Example 2</p><div class="tuf-example text-new_secondary"><p>Input: nums = [3, 4, 5, 1, 2]</p><p>Output: 3</p><p>Explanation: The original array should be [1, 2, 3, 4, 5]. So, we can notice that the array has been rotated 3 times.</p></div><br><br><p class="tuf-text-14 ">Constraints</p><div class="tuf-dark-content-box"><ul><li>&nbsp;n == nums.length</li><li>&nbsp;1 &lt;= n &lt;= 10<sup>4</sup></li><li>&nbsp;-10<sup>4</sup> &lt;= nums[i] &lt;= 10<sup>4</sup></li><li>&nbsp;All the integers of nums are unique.</li></ul></div><br><br>
