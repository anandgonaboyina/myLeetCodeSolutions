<h2><a href="https://takeuforward.org/plus/">Merge Overlapping Subintervals</a></h2><h3>Medium</h3><hr><p>Given an array of intervals where <strong>intervals[i] = [start</strong><sub><strong>i</strong></sub><strong>, end</strong><sub><strong>i</strong></sub><strong>]</strong>, merge all overlapping intervals and return an array of the<strong> non-overlapping intervals</strong> that cover all the intervals in the input.</p><p><br></p><p>You can return the intervals in any order.</p><br><br><p class="tuf-text-14">Example 1</p><div class="tuf-example text-new_secondary"><p><strong>Input: </strong>intervals = [[1,5],[3,6],[8,10],[15,18]]</p><p><strong>Output:</strong> [[1,6],[8,10],[15,18]]</p><p><strong>Explanation:</strong> Intervals [1,5] and [3,6] overlap, so they are merged into [1,6].</p></div><br><br><p class="tuf-text-14 ">Example 2</p><div class="tuf-example text-new_secondary"><p><strong>Input: </strong>intervals = [[5,7],[1,3],[4,6],[8,10]]</p><p><strong>Output:</strong> [[1,3],[4,7],[8,10]]</p><p><strong>Explanation:</strong> Intervals [4,6] and [5,7] overlap and are merged into [4,7].</p></div><br><br><p class="tuf-text-14 ">Constraints</p><div class="tuf-dark-content-box"><ul><li>1 &lt;= intervals.length &lt;= 10⁵</li><li>0 &lt;= start<sub>i</sub> &lt;= end<sub>i</sub> &lt;= 10⁵</li></ul></div><br><br>
