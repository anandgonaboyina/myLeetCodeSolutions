<h2><a href="https://takeuforward.org/plus/">4 Sum</a></h2><h3>Medium</h3><hr><p>Given an integer array <strong>nums</strong> and an integer <strong>target</strong>. Return all quadruplets [nums[a], nums[b], nums[c], nums[d]] such that:</p><p><br></p><ul><li>a, b, c, d are all distinct valid indices of nums.</li></ul><p><br></p><ul><li>nums[a] + nums[b] + nums[c] + nums[d] == target.</li></ul><p><br></p><p>Notice that the solution set must not contain duplicate quadruplets. One element can be a part of multiple quadruplets. The output and the quadruplets can be returned in any order.</p><br><br><p class="tuf-text-14">Example 1</p><div class="tuf-example text-new_secondary"><p><strong>Input</strong>: nums = [1, -2, 3, 5, 7, 9], target = 7</p><p><strong>Output</strong>: [[-2, 1, 3, 5]]</p><p><strong>Explanation</strong>:</p><p>nums[1] + nums[0] + nums[2] + nums[3] = 7</p></div><br><br><p class="tuf-text-14 ">Example 2</p><div class="tuf-example text-new_secondary"><p><strong>Input</strong>: nums = [7, -7, 1, 2, 14, 3], target = 9</p><p><strong>Output</strong>: []</p><p><strong>Explanation</strong>:</p><p>No quadruplets are present which add upto 9</p></div><br><br><p class="tuf-text-14 ">Constraints</p><div class="tuf-dark-content-box"><ul><li>1 &lt;= nums.length &lt;= 200</li><li>-10<sup>4</sup> &lt;= nums[i] &lt;= 10<sup>4</sup></li><li>-10<sup>4</sup> &lt;= target &lt;= 10<sup>4</sup></li></ul></div><br><br>
